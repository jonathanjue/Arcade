# Time-Moves-When-You-Move (Pygame)

A simple 2D top-down stealth game.

## Setup

```bash
python -m pip install -r requirements.txt
python main.py
```

## Controls
- **WASD**: move
- **R**: restart after win/lose
- **Esc**: quit

## Core mechanic
Enemies and cameras only move/rotate when you are pressing movement keys.
